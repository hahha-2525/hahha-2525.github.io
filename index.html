《几何防线》隐私政策

更新日期：2026年7月8日
生效日期：2026年7月8日

《几何防线》（以下简称“本游戏”）由“深海炸弹工作室”（以下简称“我们”）开发并运营。我们深知个人信息对您的重要性，将按照法律法规的规定，保护您的个人信息及隐私安全。本游戏是一款单机塔防游戏，但为了向您提供免费的游戏体验，我们接入了第三方广告服务。

特别提示：在您开始使用本游戏前，请务必仔细阅读并充分理解本政策。您开始使用本游戏，即表示您已充分理解并接受本政策。

一、我们如何收集和使用您的个人信息

我们根据合法、正当、必要的原则，仅会出于本政策所述的以下目的，收集和使用您的个人信息。本游戏为单机性质，我们本身不会通过游戏服务器主动收集您的个人信息，但接入的广告SDK会为了实现广告展示和效果统计而收集信息。

1. 必要信息的收集
为了保障游戏的基本运行和广告的正常展示，我们需要在您同意本政策后，获取以下必要权限和信息：

  • 设备信息：包括设备品牌、型号、操作系统版本、设备设置等，用于适配广告内容及游戏运行环境。
  • 设备标识符：如 OAID（匿名设备标识符）、Android ID，用于生成用户唯一标识、广告归因分析及防止作弊行为。
  • 网络信息：包括 IP 地址、网络类型（Wi-Fi / 5G / 4G 等），用于判断网络连接状态，确保游戏资源加载及广告内容的正常拉取。

2. 存储权限（非必要）
当您需要保存游戏进度、缓存游戏资源或保存截图时，我们会申请读取/写入外部存储的权限。您可以选择拒绝，拒绝后可能无法正常保存存档或部分资源加载变慢，但不会影响您继续游戏。

3. 例外情况（无需同意即可收集）
根据相关法律法规，在以下情形中，我们收集您的信息无需征得您的授权同意：

  • 与国家安全、国防安全直接相关的；
  • 与公共安全、公共卫生、重大公共利益直接相关的；
  • 与犯罪侦查、起诉、审判和判决执行等直接相关的；
  • 出于维护您或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；
  • 所涉及的个人信息是您自行向社会公众公开的；
  • 法律法规规定的其他情形。

二、我们使用的第三方 SDK

为了实现广告功能并维持游戏的免费运营，我们集成了以下第三方广告 SDK。这些 SDK 会独立收集您的信息，详情如下：

SDK 名称 使用目的 收集的个人信息 隐私政策链接
TapADN SDK 广告投放、监测归因、反作弊 设备品牌、型号、系统版本、OAID、Android ID、IP 地址、网络状态、应用安装列表（用于防作弊）等 点击查看 TapADN 隐私政策

关于应用列表的说明：广告 SDK 可能会检测您的应用安装列表，这是为了识别设备环境、防止广告欺诈（如模拟器检测），并非为了窥探您的隐私。

三、Cookie 及类似技术的使用

我们及第三方广告 SDK 可能使用 Cookie、像素标签等类似技术，用于：

  • 广告优化：避免您反复看到相同的广告，并向您展示可能更感兴趣的广告内容。
  • 分析服务：帮助我们分析广告的点击率和转化率，以便我们调整广告策略。

四、我们如何共享、转让、公开披露您的个人信息

1. 共享
我们不会向任何第三方公司、组织和个人分享您的个人信息，但以下情况除外：

  • 与广告合作伙伴共享：为了展示广告，我们会将您的设备标识符、IP 地址及广告交互数据共享给上述表格中提及的广告 SDK 合作伙伴。
  • 法定情形：根据法律法规、法律程序要求，或按政府主管部门的强制性要求，我们可能会对外共享您的个人信息。

2. 转让
我们不会将您的个人信息转让给任何公司、组织和个人，但涉及合并、分立、解散、被宣告破产等原因需要转移个人信息时除外。

3. 公开披露
我们不会公开披露您的个人信息，除非获得您的单独同意或法律法规另有规定。

五、我们如何存储和保护您的个人信息

1. 存储地点与期限

  • 存储地点：本游戏为单机游戏，您的游戏存档主要存储在您的设备本地。广告 SDK 收集的信息依据其隐私政策，可能存储于中国大陆或境外服务器。
  • 存储期限：我们仅在为您提供本游戏服务所必需的最短期间内保留您的个人信息。广告数据的保留期限请参考具体 SDK 的隐私政策。

2. 安全措施
我们将采用符合业界标准的安全防护措施来保护您的个人信息，包括数据加密、访问控制等技术手段。尽管我们竭尽所能，但请您理解，在互联网环境下不可能始终保证信息百分之百的安全。

六、您如何管理您的信息

在您使用本游戏期间，您可以通过以下方式管理您的信息：

  • 撤回授权：您可以在手机系统设置中关闭本游戏的存储权限，撤回您的授权。撤回后，我们可能无法继续为您提供对应的存档保存服务，但不影响您继续游戏。
  • 删除本地数据：您可以通过卸载游戏或清除游戏数据的方式，删除存储在您设备本地的所有游戏存档及缓存信息。
  • 关闭个性化广告：您可以在手机系统“隐私”或“广告”设置中，开启“限制广告跟踪”或重置广告标识符（OAID），以关闭个性化广告推荐。

七、未成年人保护

我们非常重视未成年人的个人信息保护。

  • 如果您是14 周岁以下的未成年人（儿童），在使用本游戏前，建议您和您的父母或监护人一起阅读本政策，并在征得您父母或监护人同意后再使用游戏。
  • 若您是监护人，且发现我们未经监护人同意收集了儿童个人信息，请及时联系我们，我们会尽快删除相关数据。

八、本政策如何更新

我们可能会适时更新本隐私政策。当政策发生重大变更时，我们会在游戏内通过弹窗等显著方式通知您。本政策更新后，如果您继续使用本游戏，即表示您同意接受更新后的政策。
